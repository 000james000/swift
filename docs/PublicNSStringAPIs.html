• The following are from public headers from public frameworks, which are AppKit and Foundation (verified).
  • A double dash ("--") means unavailable in Swift.
  • Creating substrings in Swift is very fast. Therefore Cocoa APIs that take NSRanges are replaced with Swift APIs that do not take ranges. One must use range-based subscripting to achieve the same effect. For example: [str doFoo:arg withRange:arg2] becomes str[arg2].doFoo(arg).
